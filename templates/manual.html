Landmark


About:

Landmark is a program that will help you explore the world map.

APIs:

Geocoder api

Geosearch api

Wiki api


Usage:

Settings:

Im [input mode] - you can switch this option to change input mode (writing or talking)
Api - you can switch this option to choose api using for search (geocoder or geosearch)

Buttons:

Help - trigger for this page
Search - used for text search
Record - used for oral search
Info - gives the information about found object
Re - used to reset search result and delete spot from map
Ho - used for focusing the map on the found object

How to search:

Cursor search:

1) click on any building
2) if spot appeared - put on info button

Text/Voice search:

1) choose input mode
2) put on search/record button
3) write/speak
4) if spot appeared - put on info button
